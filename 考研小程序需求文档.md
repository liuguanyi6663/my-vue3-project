# 校内考研小程序需求文档（最终版）

## 底部导航一：首页

### 1.1 考研倒计时（悬浮置顶）

- 支持节点：初试、复试、调剂、报名、现场确认、准考证打印
- 自动计算天 / 时 / 分 / 秒，顶部常驻，点击可展开完整时间线
- 每日一句励志文案（后台可配置）

### 1.2 广告位 / 滚动通知栏

- 轮播广告位：最多 5 张，支持点击跳转小程序页面或外部链接（需校验）
- 固定广告位：首页中部或底部，可配置图片 / 文字链
- 滚动通知栏：展示重要公告（如 “10 月 5 日预报名开始”），点击查看详情
- 后台配置：展示时长、置顶、强提醒
- 数据统计：曝光量、点击量、点击率

### 1.3 学院通知公告

- 发布权限：仅管理员（全校范围）
- 展示形式：列表页按时间倒序，置顶公告带红色 “置顶” 标签
- 已读未读标记：每条通知记录用户阅读状态，支持 “一键已读”
- 强提醒：重要通知通过微信订阅消息推送，小程序内弹窗提醒

### 1.4 今日学习计划 & 打卡

- 自定义任务：科目、任务名称、计划时长、优先级
- 打卡方式：完成任务后点击 “打卡”，可补充备注
- 打卡日历：月视图显示打卡情况
- 连续天数统计：最长连续打卡、当前连续打卡
- 微信订阅消息提醒：每日 20:00 推送未打卡提醒（需用户授权）
- 模板导入：提供基础、强化、冲刺三套模板，一键覆盖当前计划

### 1.5 快捷入口

- 本校资料中心
- 本校考研数据
- 学习打卡
- 复试工具箱（含 AI 润色）
- AI 作文批改
- 每日知识抽查
- 匿名树洞

### 1.6 考研关键时间线与自动提醒

- 时间线展示：甘特图风格，显示完整节点

- 节点列表：预报名、正式报名、现场 / 网上确认、准考证打印、初试、成绩查询、国家线、院校线、调剂开放、复试、待录取

- 提醒机制：

  - 用户可单独订阅每个节点
  - 节点前 3 天、前 1 天、当天 8:00 分别推送微信订阅消息

  

### 1.7 本校考研数据（本校特色）

#### 1.7.1 数据大屏

- 全校维度：

  - 当年报名人数、上岸人数、上岸率
  - 跨考人数、跨考率（跨专业报考人数 / 总报考人数）
  - 近 3 年趋势（上岸率 + 跨考率双折线图）

  

- 分学院维度：

  - 同上，支持按学院筛选，展示各学院报名人数、上岸人数、上岸率、跨考人数、跨考率

  

#### 1.7.2 本专业去向

- 展示近 3 年该专业学长学姐上岸去向（院校 + 人数占比）
- 分数段分布：安全分、中位分、冲刺分
- 新增跨考去向分析：本专业学生跨考最多的目标专业 TOP3 及对应上岸人数

#### 1.7.3 数据维护（后台）

- 支持 Excel 批量导入 / 导出，字段增加：

  - is_cross_major（是否跨考）
  - cross_target_major（跨考目标专业，若非跨考则为空）

  

- 支持手动修改、删除，后台自动计算全校 / 各学院跨考率

### 1.8 轻量化学习工具

#### 1.8.1 每日知识抽查

- 适用科目：英语（单词 / 词组）、政治（选择题 / 名词解释）、数学（公式 / 定理）、专业课（自定义）

- 抽查形式：

  - 英语：显示单词或中文释义，用户选择或输入对应翻译（支持四选一或手动输入）
  - 政治：随机抽取选择题或名词解释
  - 数学：展示公式 / 定理，用户填写名称或适用条件
  - 专业课：用户自定义题目，系统按艾宾浩斯曲线重复抽查

  

- 抽查规则：

  - 每日默认生成 10 道题
  - 答对 / 答错自动记录，错误题目次日重复出现
  - 支持 “仅复习错题” 模式

  

- 数据统计：每日正确率、累计掌握知识点数量

- 技术实现：英语词库、政治题库、用户自定义题目本地存储

- 提醒：微信订阅消息每日 20:00 推送 “今日抽查未完成”

#### 1.8.2 作文 / 主观题 AI 批改

- 适用场景：英语作文（大 / 小作文）、政治主观题（简答 / 论述）
- 输入方式：文本直接粘贴、图片上传（OCR 识别）
- AI 输出内容：语法纠错、词汇建议、评分、简要评语、得分点提示
- 使用限制：每日免费 5 次，单次文本上限 2000 字
- 隐私与安全：内容过微信安全 API，AI 记录仅保留 7 天
- 技术实现：调用大模型 API，设置超时与降级提示

### 1.9 复试工具箱（极简版 + AI 润色）

- 英语口语题库：30 道常见问题，提供参考回答

- 复试简历模板：3 套 Word 模板下载（带水印）

- 导师联系邮件模板：3 种场景，可一键复制

- AI 简历 / 自我介绍润色：

  - 输入：个人经历、教育背景、获奖情况等
  - 输出：中英文自我介绍，两种风格可选
  - 支持再次润色、缩短 / 扩写
  - 每日免费 3 次
  - 内容前置安全校验

  

### 1.10 学习数据导出 PDF

- 学习周报、打卡记录支持导出并打印

## 底部导航二：资料

### 2.1 本校考研资料中心（核心）

#### 2.1.1 资料分类

- 公共课：政治 / 英语 / 数学
- 专业课：按学院 → 专业 → 科目
- 本校自命题真题：年份、科目代码、科目名称
- 学长笔记、复试资料

#### 2.1.2 上传流程

1. 选择分类，填写名称、年份、科目、描述
2. 上传文件（支持 PDF、Word、图片，最大 50MB）
3. 后台审核通过后上架

#### 2.1.3 下载与预览

- 免费下载
- 预览前 5 页（PDF 格式）
- 防外传水印：下载时后端实时叠加半透明学号 + 昵称

#### 2.1.4 资料评价

- 1~5 星打分，可写短评
- 低分资料自动下沉

#### 2.1.5 收藏与文件夹

- 用户可创建自定义文件夹
- 资料可移入多个文件夹

## 五、安全合规功能（微信必需）

### 5.1 资料防外传

- 下载自动叠加学号水印
- 禁止外部分享、违规倒卖

### 5.2 版权与违规管控

- 禁止盗版资料、侵权课程、外部机构引流广告
- 一键举报，后台审核下架

### 5.3 内容风控过滤

- 关键词屏蔽广告、引流、诈骗、违规言论
- 机器初审 + 人工复核
- 全量内容过微信安全 API
- AI 生成内容前置过滤：输入与生成内容均过安全 API

## 六、后台管理系统

### 6.1 用户管理

- 角色分配：学生 / 管理员
- 违规处理：禁言、封号
- AI 使用监控：查看各用户 AI 调用次数、频率，防止滥用

### 6.2 内容审核

- 帖子、资料、广告、树洞审核
- 敏感词库、MD5 查重、微信安全 API 对接

### 6.3 资料管理

- 上传、审核、分类、水印配置
- 清理无效 / 重复资料

### 6.4 广告 / 通知管理

- 广告上下架、位置、时长、优先级配置
- 曝光、点击数据统计

### 6.5 本校数据管理

- 考研数据、光荣榜维护
- Excel 批量导入 / 导出（含跨考字段）

### 6.6 运营配置

- 轮播、通知、励志文案、时间节点管理
- AI 功能配置：每日免费次数、OCR 开关、API 密钥、超时设置

### 6.7 数据统计

- DAU/MAU、活跃时段、功能使用率
- 资料下载、发帖量 TOP10
- 按学院 / 专业使用统计
- AI 功能使用统计：调用次数、平均响应时长、用户满意度

## 七、微信小程序特别说明（技术必看）

- 登录：微信手机号授权登录，无需额外注册
- 消息提醒：使用微信订阅消息，不滥用模板消息
- 隐私合规：必须上线《隐私政策》《用户协议》
- 内容安全：所有发帖、评论、AI 输入必须走微信内容安全 API
- 资料存储：云存储 + 校内加速，水印逻辑后端实现
- AI 集成：
  - 推荐使用微信官方 AI 能力或腾讯云 AI 服务，确保合规
  - 设置请求队列和限流，避免费用超支
  - 敏感信息（如学号）不得传入 AI 模型
