# 环境配置说明

## 一、环境要求

### 1.1 硬件要求
- CPU: 双核及以上
- 内存: 4GB 及以上
- 硬盘: 10GB 可用空间

### 1.2 软件要求
| 软件 | 版本要求 | 说明 |
|------|----------|------|
| Node.js | >= 16.x | 推荐 LTS 版本 |
| npm | 随 Node.js 安装 | 或使用 yarn |
| MySQL | >= 8.0 | 需启用 utf8mb4 字符集 |
| Git | 最新版本 | 可选，用于版本控制 |

## 二、开发环境配置

### 2.1 安装 Node.js

#### Windows
1. 访问 [Node.js 官网](https://nodejs.org/)
2. 下载 LTS 版本安装包
3. 运行安装程序，按默认配置安装
4. 验证安装:
```bash
node -v
npm -v
```

#### macOS
```bash
# 使用 Homebrew 安装
brew install node

# 验证安装
node -v
npm -v
```

#### Linux (Ubuntu/Debian)
```bash
# 使用 NodeSource 仓库
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt-get install -y nodejs

# 验证安装
node -v
npm -v
```

### 2.2 安装 MySQL 8.0

#### Windows
1. 访问 [MySQL 官网](https://dev.mysql.com/downloads/mysql/)
2. 下载 MySQL Installer for Windows
3. 运行安装程序，选择 Developer Default 安装类型
4. 设置 root 用户密码（建议: 123456）
5. 配置端口为 3306
6. 启动 MySQL 服务

#### macOS
```bash
# 使用 Homebrew 安装
brew install mysql@8.0
brew services start mysql@8.0

# 设置 root 密码
mysql_secure_installation
```

#### Linux (Ubuntu/Debian)
```bash
# 更新包索引
sudo apt update

# 安装 MySQL
sudo apt install mysql-server

# 启动服务
sudo systemctl start mysql
sudo systemctl enable mysql

# 配置安全设置
sudo mysql_secure_installation
```

### 2.3 验证 MySQL 安装
```bash
# 登录 MySQL
mysql -u root -p

# 查看版本
SELECT VERSION();

# 查看字符集
SHOW VARIABLES LIKE 'character%';
SHOW VARIABLES LIKE 'collation%';

# 退出
exit;
```

## 三、项目环境配置

### 3.1 克隆/下载项目
```bash
# 如果使用 Git
git clone <repository-url>
cd my-vue3-project

# 或直接解压项目压缩包
```

### 3.2 初始化数据库
```bash
# 方式一: 直接执行 SQL 脚本
mysql -u root -p < database/schema.sql

# 方式二: 先登录 MySQL 再执行
mysql -u root -p
source e:/my-vue3-project/database/schema.sql;
```

这将创建 `kaoyan_db` 数据库及全部数据表，并插入初始数据。

### 3.3 配置后端环境

#### 3.3.1 安装依赖
```bash
cd server
npm install
```

#### 3.3.2 修改配置文件
编辑 `server/config/index.js`:

```javascript
module.exports = {
  port: process.env.PORT || 3000,
  mysql: {
    host: process.env.MYSQL_HOST || 'localhost',
    port: process.env.MYSQL_PORT || 3306,
    user: process.env.MYSQL_USER || 'root',
    password: process.env.MYSQL_PASSWORD || '123456', // 修改为你的密码
    database: process.env.MYSQL_DATABASE || 'kaoyan_db',
    timezone: '+08:00'
  },
  jwt: {
    secret: process.env.JWT_SECRET || 'kaoyan-secret-key-2024', // 生产环境请修改
    expiresIn: '7d'
  },
  upload: {
    maxFileSize: 50 * 1024 * 1024, // 50MB
    allowedTypes: ['image/jpeg', 'image/png', 'image/gif', 'application/pdf', 'application/msword', 'application/vnd.openxmlformats-officedocument.wordprocessingml.document']
  }
}
```

#### 3.3.3 使用环境变量（推荐）
创建 `server/.env` 文件:

```env
PORT=3000
MYSQL_HOST=localhost
MYSQL_PORT=3306
MYSQL_USER=root
MYSQL_PASSWORD=123456
MYSQL_DATABASE=kaoyan_db
JWT_SECRET=your-secret-key-here-change-in-production
```

#### 3.3.4 启动后端服务
```bash
# 开发模式（自动重启）
npm run dev

# 生产模式
npm start
```

后端服务将运行在 `http://localhost:3000`

### 3.4 配置前端环境

#### 3.4.1 安装依赖
```bash
cd client
npm install
```

#### 3.4.2 修改 API 地址
编辑 `client/src/api/request.js`，修改 `BASE_URL`:

```javascript
// 开发环境
const BASE_URL = 'http://localhost:3000/api'

// 生产环境
// const BASE_URL = 'https://your-domain.com/api'
```

#### 3.4.3 启动前端服务

##### H5 模式（浏览器开发）
```bash
npm run dev:h5
```
访问 `http://localhost:5173` 预览

##### 微信小程序模式
```bash
npm run dev:mp-weixin
```
编译完成后，打开微信开发者工具，导入 `client/dist/dev/mp-weixin` 目录

## 五、常见问题

### 5.1 MySQL 连接失败
**问题**: `Error: connect ECONNREFUSED 127.0.0.1:3306`

**解决方案**:
1. 检查 MySQL 服务是否启动
```bash
# Windows
net start MySQL80

# macOS
brew services list

# Linux
sudo systemctl status mysql
```
2. 检查端口是否正确
3. 检查用户名和密码是否正确

### 5.2 端口被占用
**问题**: `Error: listen EADDRINUSE: address already in use :::3000`

**解决方案**:
```bash
# Windows 查看占用端口的进程
netstat -ano | findstr :3000

# 结束进程
taskkill /PID <进程ID> /F

# macOS/Linux
lsof -ti:3000 | xargs kill -9
```

### 5.3 npm install 失败
**解决方案**:
```bash
# 清理 npm 缓存
npm cache clean --force

# 使用淘宝镜像
npm config set registry https://registry.npmmirror.com

# 重新安装
npm install
```

### 5.4 数据库字符集问题
**问题**: 中文字符显示乱码

**解决方案**:
确保 MySQL 配置文件 (`my.cnf` 或 `my.ini`) 包含:
```ini
[client]
default-character-set=utf8mb4

[mysql]
default-character-set=utf8mb4

[mysqld]
character-set-server=utf8mb4
collation-server=utf8mb4_unicode_ci
```

## 六、生产环境部署

### 6.1 服务器要求
- OS: Linux (Ubuntu 20.04+ / CentOS 8+)
- Node.js: 18.x LTS
- MySQL: 8.0
- Nginx: 最新稳定版

### 6.2 部署步骤
1. 安装 Node.js、MySQL、Nginx
2. 克隆项目到服务器
3. 配置环境变量（使用 .env 文件）
4. 初始化数据库
5. 安装依赖并构建
6. 使用 PM2 管理 Node.js 进程
7. 配置 Nginx 反向代理
8. 配置 HTTPS（使用 Let's Encrypt）

### 6.3 PM2 配置
```bash
# 安装 PM2
npm install -g pm2

# 启动服务
cd server
pm2 start app.js --name kaoyan-server

# 查看状态
pm2 status

# 查看日志
pm2 logs kaoyan-server

# 开机自启
pm2 startup
pm2 save
```
